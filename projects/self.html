bbo.do (biːˈbodo)

bbo.do is my personal website where I share projects, notes, and experiments.

About #

This site serves as a central hub for my creative and technical work, featuring:

  • Projects: Interactive experiments and tools
  • Notes: Thoughts on web development, art, music, and transdisciplinary topics. Becoming more and more interactive, might merge with projects.

Tech Stack #

Built with modern web technologies:

  • Vue.js
  • Vite
  • UnoCSS

Philosophy #

It’s designed to be:

  • Fast: Optimized performance with minimal overhead
  • Open: The source code is available under MIT license, content under CC BY-NC-SA 4.0

The domain name bbo.do cleverly combines my initials (bbo) with the .do TLD, creating a play on words that sounds like "be-bodo" - a nod to my name Bodo Brägger.
